# Powerwash Company Website

## Description:
This project is a website designed for a powerwashing company to showcase its services, include customer testimonials, and provide easy access for potential clients to contact the company.

## Features:
- Home page with service overview
- About us section
- Services offered
- Testimonials from customers
- Contact form for inquiries

## Technologies Used:
- HTML
- CSS
- JavaScript

## Installation:
To run this project locally, follow these steps:
1. Clone the repository: `git clone https://github.com/nocksneamru-byte/powerwash-nwi.git`
2. Open the `index.html` file in your web browser.
